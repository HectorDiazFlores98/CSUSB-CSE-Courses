This website has been inspired by my favorite football team, Manchester United.
I used multiple outside sources such as images and widgets to help create this website.

The index.html page features
  • A brief history of Manchester United
  • An embedded Twitter feed linked to the Official Man Utd account

The Squad.html page features
  • Player Portrait
  • Player First Name
  • Player Last Name
  • Player Position
  • Player Number
  • Links to all of the players Wiki page

The League.html page features
  • A brief history of the Premier League
  • A live, self updating table of the Premier Leagues team positions

The About.html page features
  • Information about the page

All of the pages include
  • The Nav Bar with links to squad, league and About information
  • To access the index page again, click on the "Manchester United" header on the top left

For this website the outside resources I used are
  • Background
  • An embedded twitter feed using the twitter developer tools
  • The player portraits are found in the Premier League AWS service page
  • Information and paragraphs about the team are from the Manchester United F.C. Wiki Page
  • The Premier League table widget is a link from FCTables.com
